# 🛡️ Security Hub – Cyber Fraud Awareness Website

A modern and informative **Cybersecurity Awareness Website** designed to educate users about online scams, frauds, and digital safety practices. Built with a focus on **real-world threats and prevention techniques**.

---

## 🔥 Project Overview

**Security Hub** is a multi-page web application that spreads awareness about various cyber frauds such as phishing, scams, and social engineering attacks. It provides users with practical tips to stay safe online.

---

## ✨ Features

✔️ Multi-page responsive website
✔️ Detailed information on cyber fraud types
✔️ Real-world scam examples and case studies
✔️ Prevention tips and safety guidelines
✔️ “Report Scam” section
✔️ Contact form for user interaction
✔️ Modern UI with clean design

---

## 🌐 Pages Included

* 🏠 Home (Overview of cybersecurity awareness)
* ⚠️ Types of Scams
* 📂 Real Cases
* 💡 Safety Tips
* 🚨 Report Scam
* 📞 Contact Page
